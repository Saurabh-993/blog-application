So Here we are going to Create a Full Blog Application.

Tech Stack We Are Going to use is :
-- MongoDB as the Database for our Application.
-- nodeJS as the Server for our backend 
    - mongoose, express, dotenv, helmet,argon2.
    - Multer for files uploading (mainly banner and profile)
    - nodemon as for our development tool.
-- Tailwind CSS and Figma for designing our Frontend from wireframe to Code.
-- JWT for the tokens verification and authorization part.
-- we are going to use DOM manipulation and Fetch API for req. making.
-- Also we will respect the principle of Sepration of Concern (SoC).
